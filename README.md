# Image Gallery

## 📌 Project Overview

This project is a responsive Image Gallery built using **HTML, CSS, and JavaScript**. It displays images in a clean grid layout and allows users to browse them using a lightbox with Next and Previous navigation. The gallery includes smooth animations, hover effects, and a mobile-friendly design.

## ✨ Features

* Responsive image gallery layout
* Lightbox image viewer
* Next and Previous navigation
* Smooth transitions and animations
* Hover effects on images
* Mobile-friendly and responsive design
* Clean and modern user interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

```
Image-Gallery/
│── index.html
│── style.css
│── script.js
└── images/
    ├── image1.jpg
    ├── image2.jpg
    ├── image3.jpg
    └── ...
```

## 🚀 How to Run the Project

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` in your web browser.
4. Click on any image to open the lightbox.
5. Use the Next and Previous buttons to navigate through the images.
